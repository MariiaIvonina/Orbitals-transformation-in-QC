# IAO/IBO Analysis for Nucleic Acid Systems

This repository accompanies the chapter:

"Chemical Bond Analysis in Biomolecular Systems Using Intrinsic Atomic and Bond Orbitals (IAOs and IBOs) with PySCF"

Methods in Molecular Biology (Springer Nature).

The notebook demonstrates how to compute:

- IAOs
- IBOs
- Mulliken populations in IAO basis
- Mayer bond indices
- IBO center classification

using PySCF and visualize produces orbitals.

To setup conda environment with required packages, you can just run
```
conda env create -f environment.yml
```
